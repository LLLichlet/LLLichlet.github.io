1.2 Subspace

Definition 1.2.1.

Let V be a vector space over a field F. A subset U⊆V is a subspace if

  • •

    0∈U,

  • •

    u,v∈U implies u+v∈U,

  • •

    λ∈F, u∈U implies λ⁢u∈U.

Definition 1.2.2.

Let {Ui}i∈I be a family of subspaces of V. The sum ∑i∈IUi is the set of all finite sums

ui1+⋯+uik,k≥0,ij∈I,uij∈Uij,

where the empty sum (k=0) is understood as 0.

Proposition 1.2.3.

∑i∈IUi is a subspace of V, and it is the smallest subspace containing every Ui.

Proof.

Each Ui is contained in the sum (take k=1). The sum is a subspace: 0 is the empty sum; the sum of two finite sums is a finite sum; scalar multiplication distributes over a finite sum. If W is a subspace containing all Ui, then W contains every finite sum of their elements, hence ∑i∈IUi⊆W. ∎

Definition 1.2.4.

The sum ∑i∈IUi is a direct sum, denoted ⨁i∈IUi, if every vector in the sum admits a unique representation ui1+⋯+uik with uij∈Uij and ij distinct.

Proposition 1.2.5.

∑i∈IUi is a direct sum if and only if whenever

∑i∈Iui=0,ui∈Ui⁢ (only finitely many nonzero),

then ui=0 for all i∈I.

Proof.

If the sum is direct, 0 has the unique representation 0=∑i0, so all ui=0. Conversely, if ∑iui=∑ivi, then ∑i(ui−vi)=0, hence ui−vi=0 for all i, giving uniqueness. ∎

Proposition 1.2.6.

Let U,W⊆V be subspaces. The sum U+W is a direct sum if and only if U∩W={0}.

Proof.

If v∈U∩W is nonzero, then v=v+0=0+v gives two distinct representations. Conversely, if U∩W={0} and u+w=u′+w′, then u−u′=w′−w∈U∩W={0}, so u=u′ and w=w′. ∎
