1.3 Linear Maps

Definition 1.3.1.

Let V, W be vector spaces over a field F. A map f:V→W is linear if

  • •

    f⁢(u+v)=f⁢(u)+f⁢(v) for all u,v∈V,

  • •

    f⁢(λ⁢v)=λ⁢f⁢(v) for all λ∈F, v∈V.

Definition 1.3.2.

For a linear map f:V→W, the kernel and image are

ker⁡f={v∈V∣f⁢(v)=0},im⁡f={f⁢(v)∣v∈V}.
Proposition 1.3.3.

ker⁡f is a subspace of V, and im⁡f is a subspace of W.

Proof.

For ker⁡f: f⁢(0)=0, so 0∈ker⁡f. If u,v∈ker⁡f, then f⁢(u+v)=f⁢(u)+f⁢(v)=0, so u+v∈ker⁡f. If v∈ker⁡f and λ∈F, then f⁢(λ⁢v)=λ⁢f⁢(v)=0, so λ⁢v∈ker⁡f.

For im⁡f: 0=f⁢(0)∈im⁡f. If f⁢(u),f⁢(v)∈im⁡f, then f⁢(u)+f⁢(v)=f⁢(u+v)∈im⁡f. If f⁢(v)∈im⁡f and λ∈F, then λ⁢f⁢(v)=f⁢(λ⁢v)∈im⁡f. ∎

Proposition 1.3.4.

A linear map f:V→W is injective if and only if ker⁡f={0}.

Proof.

If f is injective and f⁢(v)=0=f⁢(0), then v=0, so ker⁡f={0}. Conversely, if ker⁡f={0} and f⁢(u)=f⁢(v), then f⁢(u−v)=0, so u−v∈ker⁡f={0}, hence u=v. ∎
