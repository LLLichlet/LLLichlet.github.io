1.1 Definition

Definition 1.1.1.

Let F be a field. A vector space over F is a set V equipped with two operations

+ :V×V→V,(u,v)↦u+v,
⋅ :F×V→V,(λ,v)↦λ⁢v,

satisfying the following axioms:

  1. 1.

    (V,+) is an abelian group. That is, for all u,v,w∈V,

    • •

      (u+v)+w=u+(v+w),

    • •

      there exists 0∈V such that v+0=v,

    • •

      for each v∈V there exists −v∈V such that v+(−v)=0,

    • •

      u+v=v+u.

  2. 2.

    Scalar multiplication is compatible with field operations: for all λ,μ∈F and u,v∈V,

    • •

      λ⁢(u+v)=λ⁢u+λ⁢v,

    • •

      (λ+μ)⁢v=λ⁢v+μ⁢v,

    • •

      (λ⁢μ)⁢v=λ⁢(μ⁢v),

    • •

      1⁢v=v, where 1 is the multiplicative identity of F.

Elements of V are called vectors; elements of F are called scalars.

Definition 1.1.2.

Let S be any set and F a field. The set

FS={f∣f:S→F}

of all functions from S to F is a vector space over F with operations defined pointwise:

(f+g)⁢(x) =f⁢(x)+g⁢(x),
(λ⁢f)⁢(x) =λ⋅f⁢(x),

for all f,g∈FS, λ∈F, and x∈S.

Proposition 1.1.3.

FS is a vector space over F.

Proof.
