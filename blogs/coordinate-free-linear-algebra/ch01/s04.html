1.4 Quotient Space

Definition 1.4.1.

Let V be a vector space over a field F and U⊆V a subspace. For v∈V, the coset of v modulo U is

v+U={v+u∣u∈U}.

The quotient space V/U is the set of all cosets

V/U={v+U∣v∈V},

with addition and scalar multiplication defined by

(v+U)+(w+U)=(v+w)+U,λ⁢(v+U)=λ⁢v+U,

for v,w∈V and λ∈F.

Definition 1.4.2.

The quotient map π:V→V/U is defined by

π⁢(v)=v+U.
Proposition 1.4.3.

π is a linear map, ker⁡π=U, and im⁡π=V/U.

Proof.

Left as an exercise. ∎

Proposition 1.4.4.

V/U is a vector space over F.

Proof.

Left as an exercise. ∎

Proposition 1.4.5.

For any linear map f:V→W,

V/ker⁡f≅im⁡f.
Proof.

Define Φ:V/ker⁡f→im⁡f by Φ⁢(v+ker⁡f)=f⁢(v).

Well-defined: if v+ker⁡f=w+ker⁡f, then v−w∈ker⁡f, so f⁢(v)−f⁢(w)=f⁢(v−w)=0, hence f⁢(v)=f⁢(w).

Φ is linear: Φ⁢((v+ker⁡f)+(w+ker⁡f))=Φ⁢((v+w)+ker⁡f)=f⁢(v+w)=f⁢(v)+f⁢(w)=Φ⁢(v+ker⁡f)+Φ⁢(w+ker⁡f), and similarly for scalar multiplication.

Injective: if Φ⁢(v+ker⁡f)=0, then f⁢(v)=0, so v∈ker⁡f, hence v+ker⁡f=0+ker⁡f.

Surjective: for any f⁢(v)∈im⁡f, we have Φ⁢(v+ker⁡f)=f⁢(v). ∎
